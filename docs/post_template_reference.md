## 待確認：自動化的資料落差（重要）

範本裡的這些資訊，目前的爬蟲/瀏覽流程**拿不到**：
- 官網原價 vs 折扣後價格的對比（例：原價$1190 官網價$1100）
- 折扣代碼（例：BMM910）
- 商品詳細規格（產地、重量、包裝方式）
- 保存/配送方式
- 品牌/店家介紹段落
- 會員贈點、滿額贈禮等活動細節

這些是店家自己提供的完整促銷素材（通常來自美安/店家的官方文案），
不是 hot-deals / daily-deals / our-brands 頁面上那種一句話促銷文案能涵蓋的。

依照專案規則「不可編造資料」，自動化流程只能使用**實際抓得到的欄位**
（商品名稱、價格、簡短促銷文案、圖片、連結、現金回饋/IBV%），
不能虛構折扣碼、規格、保存方式等內容。
